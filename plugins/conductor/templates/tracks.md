## 轨道创建清单

创建新轨道时：

1. [ ] 在此注册表中添加条目
2. [ ] 创建轨道目录：`./tracks/{{track-id}}/`
3. [ ] 根据模板创建 spec.md
4. [ ] 根据模板创建 plan.md
5. [ ] 根据模板创建 metadata.json
6. [ ] 更新 index.md，添加新轨道引用

## 注意事项

- 轨道 ID 应使用小写字母和连字符（例如 `user-auth`、`api-v2`）
- 保持描述简洁（单行）
- 轨道优先级分为：critical（关键）、high（高）、medium（中）、low（低）
- 每季度归档已完成的轨道
