# Team Status (团队状态)

显示活动智能体团队的当前状态,包括成员、任务和进度。

## 阶段 1:团队发现

1. 从 `$ARGUMENTS` 解析团队名称和标志:
   - 如果提供了团队名称,直接使用
   - 如果没有团队名称,检查 `~/.claude/teams/` 中的活动团队
   - 如果存在多个团队且未指定名称,列出所有团队并询问用户选择
   - `--tasks`: 仅显示任务详细信息
   - `--members`: 仅显示成员详细信息
   - `--json`: 输出原始 JSON 而不是格式化表格

2. 使用 Read 工具从 `~/.claude/teams/{team-name}/config.json` 读取团队配置
3. 调用 `TaskList` 获取当前任务状态

## 阶段 2:状态显示

### 成员表

显示每个团队成员及其当前状态:

```
团队: {team-name}
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

成员:
  名称              角色              状态
  ─────────────────────────────────────────
  security-rev      team-reviewer     正在处理任务 #2
  perf-rev          team-reviewer     空闲
  arch-rev          team-reviewer     正在处理任务 #4
```

### 任务表

显示带有状态、受托人和依赖关系的任务:

```
任务:
  ID   状态        所有者           主题
  ─────────────────────────────────────────────────
  #1   已完成       security-rev    审查认证模块
  #2   进行中       security-rev    审查 API 端点
  #3   已完成       perf-rev        分析数据库查询
  #4   进行中       arch-rev        分析模块结构
  #5   待处理       (未分配)        整合发现

进度: 40% (2/5 已完成)
```

### JSON 输出

如果设置了 `--json` 标志,将原始团队配置和任务列表作为 JSON 输出。
