# 初创企业指标框架

从种子轮到A轮，针对不同初创企业商业模式的关键绩效指标跟踪、计算和优化的综合指南。

## 概述

在正确的阶段跟踪正确的指标。专注于对融资和运营卓越重要的单位经济效益、增长效率和现金管理指标。

## 通用初创企业指标

### 收入指标

**MRR（月度经常性收入）**

```
MRR = Σ (活跃订阅数 × 月度价格)
```

**ARR（年度经常性收入）**

```
ARR = MRR × 12
```

**增长率**

```
MoM 增长率 = (本月MRR - 上月MRR) / 上月MRR
YoY 增长率 = (本年ARR - 去年ARR) / 去年ARR
```

**目标基准：**

- 种子轮：15-20% 月度增长率
- A轮：10-15% 月度增长率，3-5倍 年度增长率
- B轮及以后：100%+ 年度增长率（40法则）

### 单位经济效益

**CAC（客户获取成本）**

```
CAC = 总销售营销支出 / 新获取客户数
```

包括：销售薪资、营销支出、工具、管理费用

**LTV（客户生命周期价值）**

```
LTV = ARPU × 毛利率% × (1 / 流失率)
```

简化版本：

```
LTV = ARPU × 平均客户生命周期 × 毛利率%
```

**LTV:CAC 比率**

```
LTV:CAC = LTV / CAC
```

**基准：**

- LTV:CAC > 3.0 = 健康
- LTV:CAC 1.0-3.0 = 需要改进
- LTV:CAC < 1.0 = 不可持续

**CAC 回收期**

```
CAC 回收期 = CAC / (ARPU × 毛利率%)
```

**基准：**

- < 12个月 = 优秀
- 12-18个月 = 良好
- > 24个月 = 令人担忧

### 现金效率指标

**烧钱率**

```
月度烧钱率 = 月度收入 - 月度支出
```

负烧钱率 = 亏损（早期阶段典型情况）

**跑道**

```
跑道（月数） = 现金余额 / 月度烧钱率
```

**目标：** 始终保持12-18个月的跑道

**烧钱乘数**

```
烧钱乘数 = 净烧钱额 / 净新增ARR
```

**基准：**

- < 1.0 = 卓越效率
- 1.0-1.5 = 良好
- 1.5-2.0 = 可接受
- > 2.0 = 低效

越低越好（花费更少来产生ARR）

## SaaS指标

### 收入构成

**新增MRR**
新客户 × ARPU

**扩张MRR**
现有客户的增购和交叉销售

**收缩MRR**
现有客户的降级

**流失MRR**
流失客户

**净新增MRR公式：**

```
净新增MRR = 新增MRR + 扩张MRR - 收缩MRR - 流失MRR
```

### 留存指标

**客户留存率**

```
客户留存率 = (期末客户数 - 新增客户数) / 期初客户数
```

**金额留存率（NDR - 净金额留存率）**

```
NDR = (期初ARR + 扩张 - 收缩 - 流失) / 期初ARR
```

**基准：**

- NDR > 120% = 业内领先
- NDR 100-120% = 良好
- NDR < 100% = 需要改进

**毛留存率**

```
毛留存率 = (期初ARR - 流失 - 收缩) / 期初ARR
```

**基准：**

- > 90% = 优秀
- 85-90% = 良好
- < 85% = 令人担忧

### SaaS特定指标

**神奇数字**

```
神奇数字 = 净新增ARR（季度） / 销售营销支出（上一季度）
```

**基准：**

- > 0.75 = 高效，准备规模化
- 0.5-0.75 = 中等效率
- < 0.5 = 低效，暂不规模化

**40法则**

```
40法则 = 收入增长率% + 利润率%
```

**基准：**

- > 40% = 优秀
- 20-40% = 可接受
- < 20% = 需要改进

**示例：**
50%增长率 + (-10%)利润率 = 40% ✓

**快速比率**

```
快速比率 = (新增MRR + 扩张MRR) / (流失MRR + 收缩MRR)
```

**基准：**

- > 4.0 = 健康增长
- 2.0-4.0 = 中等
- < 2.0 = 存在流失问题

## 市场指标

### GMV（商品交易总额）

**总交易量：**

```
GMV = Σ (交易价值)
```

**增长率：**

```
GMV 增长率 = (本期GMV - 上期GMV) / 上期GMV
```

**目标：** 早期阶段20%+ 月度增长率

### 抽成率

```
抽成率 = 净收入 / GMV
```

**典型范围：**

- 支付处理器：2-3%
- 电商平台：10-20%
- 服务市场：15-25%
- 高价值B2B：5-15%

### 市场流动性

**成交时间**
从上架到销售/匹配需要多长时间？

**成交率**
成功交易占总请求的百分比

**复购率**
多次交易用户的百分比

**基准：**

- 成交率 > 80% = 强流动性
- 复购率 > 60% = 强留存

### 市场平衡

**供需比率：**
跟踪供需双方的相对增长。

**警告信号：**

- 供给过多：成交率低，供应商受挫
- 需求过多：等待时间长，客户受挫

**目标：** 平衡增长（1:1比率理想，但因模式而异）

## 消费者/移动端指标

### 参与度指标

**DAU（日活跃用户）**
每天活跃的唯一用户数

**MAU（月活跃用户）**
每月活跃的唯一用户数

**DAU/MAU 比率**

```
DAU/MAU = DAU / MAU
```

**基准：**

- > 50% = 卓越（日常习惯）
- 20-50% = 良好
- < 20% = 参与度弱

**会话频率**
每个用户每天/周的平均会话次数

**会话时长**
每次会话的平均时间

### 留存曲线

**第1天留存：** 次日返回用户的百分比
**第7天留存：** 注册后7天活跃用户的百分比
**第30天留存：** 注册后30天活跃用户的百分比

**基准（第30天）：**

- > 40% = 优秀
- 25-40% = 良好
- < 25% = 较弱

**留存曲线形状：**

- 趋于平缓的曲线 = 良好（用户形成习惯）
- 急剧下降 = 产品市场契合度差

### 病毒系数（K因子）

```
K因子 = 每用户邀请数 × 邀请转化率
```

**示例：**
10次邀请/用户 × 20%转化率 = 2.0 K因子

**基准：**

- K > 1.0 = 病毒式增长
- K = 0.5-1.0 = 强推荐
- K < 0.5 = 病毒传播力弱

## B2B指标

### 销售效率

**赢单率**

```
赢单率 = 赢单数 / 总机会数
```

**目标：** 新销售团队20-30%，成熟团队30-40%

**销售周期长度**
从机会到成交的平均天数

**越短越好：**

- 中小企业：30-60天
- 中端市场：60-120天
- 企业级：120-270天

**平均合同价值（ACV）**

```
ACV = 总合同价值 / 合同期限（年）
```

### 销售漏斗指标

**漏斗覆盖率**

```
漏斗覆盖率 = 总漏斗价值 / 配额
```

**目标：** 3-5倍覆盖率（需要3-5倍漏斗来达成配额）

**各阶段转化率：**

- 线索 → 机会：10-20%
- 机会 → 演示：50-70%
- 演示 → 提案：30-50%
- 提案 → 成交：20-40%

## 分阶段指标

### 种子前（产品市场契合度）

**关注指标：**

1. 活跃用户增长
2. 用户留存（第7天、第30天）
3. 核心参与度（会话、使用的功能）
4. 定性反馈（NPS、访谈）

**无需担心：**

- 收入（可能为零）
- CAC（尚未优化）
- 单位经济效益

### 种子轮（50万-200万美元ARR）

**关注指标：**

1. MRR增长率（15-20% 月度）
2. CAC和LTV（建立基线）
3. 毛留存率（> 85%）
4. 核心产品参与度

**开始跟踪：**

- 销售效率
- 烧钱率和跑道

### A轮（200万-1000万美元ARR）

**关注指标：**

1. ARR增长（3-5倍 年度）
2. 单位经济效益（LTV:CAC > 3，回收期 < 18个月）
3. 净金额留存率（> 100%）
4. 烧钱乘数（< 2.0）
5. 神奇数字（> 0.5）

**成熟跟踪：**

- 40法则
- 销售效率
- 漏斗覆盖率

## 指标跟踪最佳实践

### 数据基础设施

**要求：**

- 单一数据源（分析平台）
- 实时或每日更新
- 自动化计算
- 历史跟踪

**工具：**

- Mixpanel、Amplitude（产品分析）
- ChartMogul、Baremetrics（SaaS指标）
- Looker、Tableau（BI仪表板）

### 报告节奏

**每日：**

- MRR、活跃用户
- 注册、转化

**每周：**

- 增长率
- 留存队列
- 销售漏斗

**每月：**

- 完整指标套件
- 董事会报告
- 投资者更新

**每季度：**

- 趋势分析
- 基准对比
- 战略回顾

### 常见错误

**错误1：虚荣指标**
不要关注：

- 总用户数（无留存）
- 页面浏览量（无参与度）
- 下载量（无激活）

专注于与价值相关的可操作指标。

**错误2：指标过多**
深入跟踪5-7个核心指标，而非肤浅跟踪50个。

**错误3：忽视单位经济效益**
CAC和LTV即使在种子轮阶段也至关重要。

**错误4：不分群**
按客户细分、渠道、队列分解指标。

**错误5：操纵指标**
优化真实的业务成果，而非仪表板数字。

## 投资者指标

### 风投机构想看什么

**种子轮：**

- MRR增长率
- 用户留存
- 早期单位经济效益
- 产品参与度

**A轮：**

- ARR和增长率
- CAC回收期 < 18个月
- LTV:CAC > 3.0
- 净金额留存率 > 100%
- 烧钱乘数 < 2.0

**B轮及以后：**

- 40法则 > 40%
- 高效增长（神奇数字）
- 盈利路径
- 市场领导地位指标

### 指标展示

**仪表板格式：**

```
当前MRR：$250K（↑ 18% 月度）
ARR：$3.0M（↑ 280% 年度）
CAC：$1,200 | LTV：$4,800 | LTV:CAC = 4.0x
NDR：112% | 客户留存率：92%
烧钱率：$18万/月 | 跑道：18个月
```

**包括：**

- 当前值
- 增长率或趋势
- 背景（目标、基准）

## 快速开始

要实施初创企业指标框架：

1. **确定商业模式** - SaaS、市场、消费者、B2B
2. **选择5-7个核心指标** - 基于阶段和模式
3. **建立跟踪** - 设置分析和仪表板
4. **计算单位经济效益** - CAC、LTV、回收期
5. **设定目标** - 使用基准作为目标
6. **定期审查** - 每周审查核心指标
7. **与团队分享** - 对齐目标和进度
8. **更新投资者** - 每月/每季度报告

有关详细定义、基准和示例，请参阅 `references/` 和 `examples/`。
