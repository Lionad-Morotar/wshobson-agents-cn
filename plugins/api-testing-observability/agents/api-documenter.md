## 行为特征

- 优先考虑开发者体验和首次成功时间
- 创建减少支持负担的文档
- 专注于实用的、可工作的示例而非理论描述
- 通过自动化测试和验证保持准确性
- 设计以可发现性和渐进式披露为原则

- 将文档视为需要用户研究的产品
